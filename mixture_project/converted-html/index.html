A Word From The Editor

P2 or Pillar 2 within ThoughtWorks represents software excellence. P1 is sustainable business and P3, social justice. These three equally support ThoughtWorks. This magazine covers software excellence within ThoughtWorks.

Software excellence evades definition. It’s easy to think of concise lines of code laid out like prose. But we didn’t want this magazine to be exclusively for those who think in code. ThoughtWorks is filled with testers, designers, innovators and analysts. We needed to push any definition further to consider the excellence that happens before we even consider the solution, through continuous design, integration, deployment and delivery.

It was the 7th of May when we embarked on this project. Myself and Mike Gardiner talking about it over coffee. We decided we release monthly without regard for getting it ‘right’. We’ll release, get feedback, learn, improve before doing it all again. This lightweight, do, learn, adapt is at the core of how ThoughtWorks operates and comes to life in Big data ain’t so big while in Uncovering your data’s dark matter Jen Smith looks at the learning side.

In order to learn we need feedback. So tell us what you think p2@thoughtworks.com

Welcome to the inaugural issue of P2 magazine.
Ryan Boucher, Editor in Chief
